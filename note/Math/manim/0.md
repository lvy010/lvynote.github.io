# Manim docs：数学动画引擎

Manim是一个专为**数学可视化**设计的**强大动画引擎**

允许用户在**场景**(主画布)上==定义称为**Mobjects**的*可视化元素*==，通过**动画**进行动态展示，最终将这些动画渲染成视频。该系统包含以下核心组件：

1. **摄像机系统** - 控制观察视角
2. **着色器封装器** - 实现高效GPU渲染
3. **配置系统** - 提供灵活的自定义选项
4. **渲染与交互循环** - 驱动整个动画流程
