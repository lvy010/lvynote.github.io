# docs：Avoid-MPC

`Avoid-MPC`项目为==无人机配备了**自主避障**能力==。

它利用*高级模型预测控制（MPC）*系统进行最优路径规划，并通过*实时KD树地图*实现即时障碍物感知。该系统通过ROS与*AirSim模拟器*无缝集成，支持对模拟无人机的全面测试与控制

能处理传感器数据、管理配置，并生成精确的控制指令，以实现安全高效的导航
