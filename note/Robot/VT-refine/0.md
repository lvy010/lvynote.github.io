# docs：VT-Refine项目

VT-Refine项目就像一个**机器人训练学院**，教会机器人如何执行*复杂的双手装配任务*。

通过结合从真实*仿真环境*中收集的*视觉与触觉反馈*来实现这一目标。训练过程包含基于大规模数据集的预训练阶段，以及后续的微调过程，使机器人能够==在虚拟世界中掌握==*精确交互*和复杂的手-眼-触觉协调能力。
