# docs：DeepSeek-OCR

DeepSeek-OCR 是一个强大的系统，用于对文档和图像执行**光学字符识别（OCR）**。它就像一个智能助手，接收你的*图像和文本提示*（如"将文档转换为markdown"），然后深度*分析视觉内容*并将其与文本提示结合。最终，它利用大语言模型从文档中*生成准确且连贯的文本输出*，同时通过vLLM框架优化速度和效率。
