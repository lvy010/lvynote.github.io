# docs：Claude Code

`Claude Code`是一款**智能编程助手**，常驻于终端环境，是开发者的*智能伙伴*。它能通过自然语言指令**自动化复杂编程和Git操作**，并与*GitHub问题管理系统*深度集成，显著提升开发效率。
