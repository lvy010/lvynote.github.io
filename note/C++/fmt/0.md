# docs：fmt

`fmt` 项目是一个**现代 C++ 格式化库**，专为*速度、安全性和易用性*而设计

它允许开发者使用灵活的、类似 Python 的语法==从各种数据类型创建格式化字符串==。主要功能包括*编译时格式字符串验证*、高效处理参数，以及优化输出到不同目标（如控制台或文件）。
