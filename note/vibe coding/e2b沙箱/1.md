# docs：E2B

E2B提供了一套*开源基础设施*，允许我们在**安全隔离的云沙箱**中运行AI生成的代码

开发者通过**SDK**（JavaScript/Python）*远程控制*这些沙箱，执行命令并管理文件

**沙箱模板**作为*可定制的蓝图*定义执行环境，便于快速部署专为特定AI任务优化的沙箱。
